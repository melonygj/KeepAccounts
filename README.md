# 记账本 App

一款简洁实用的个人记账应用，支持多账户管理、分类统计、预算管理和借贷记录。

## 功能特性

### 核心功能
- 📝 **快速记账** - 支持收入、支出、转账记录
- 💰 **多账户管理** - 现金、银行卡、支付宝、微信等
- 📊 **数据统计** - 月度收支分析
- 💳 **借贷管理** - 借入借出记录
- 📈 **预算设置** - 控制消费

### 账户类型
- 现金
- 银行卡
- 信用卡
- 支付宝
- 微信
- 投资账户

### 默认分类
**支出**: 餐饮、交通、购物、娱乐、医疗、教育、住房、通讯、其他

**收入**: 工资、奖金、投资收益、兼职、其他收入

## 技术栈

- **语言**: Kotlin
- **UI**: Jetpack Compose + Material 3
- **数据库**: Room
- **架构**: MVVM
